## install

[download PDFtk Server](https://www.pdflabs.com/tools/pdftk-server/)

```bash
git clone https://github.com/data-doge/simple-immigration.git
cd simple-immigration
npm install
```

## develop

```bash
npm run develop
```

browse to http://localhost:3000/
